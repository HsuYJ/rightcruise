# rightcruise
rightCruise.js makes it possible to scroll via mousedown mouse right button in anywhere of web page.

# Usage
<p>Download and Include rightCruise.js.</p>
```
<script type="text/javascript" src="path/to/rightCruise.js"></script>
```
<p>Done.</p>
